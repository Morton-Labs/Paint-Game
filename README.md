# Paint Game

A Python drawing application built with Turtle Graphics.

## Features
- Draw lines, squares, circles, rectangles and triangles
- Switch colors with keyboard keys

## Controls
| Key | Action |
|-----|--------|
| K | Black |
| W | White |
| G | Green |
| B | Blue |
| R | Red |
| P | Purple |
| l | Line |
| s | Square |
| c | Circle |
| r | Rectangle |
| t | Triangle |
| u | Undo |
